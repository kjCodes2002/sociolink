# SocioLink

A centralized social profile management dashboard that allows users to manage and share all their social media handles from a single, elegant interface.

## üåü Overview

SocioLink simplifies online identity management by providing a unified platform where users can organize, manage, and share their social media presence. Instead of juggling multiple profiles across different platforms, users can create a single hub that showcases all their social links in one beautiful interface.

## ‚ú® Features

- **Centralized Dashboard**: Manage all your social media handles from one place
- **Easy Sharing**: Share your complete online presence with a single link
- **Modern UI**: Clean, responsive design built with Tailwind CSS and shadcn/ui
- **Real-time Sync**: Instant updates across all devices using Firestore
- **Secure Authentication**: Firebase-powered user authentication and authorization
- **Profile Customization**: Upload and manage profile images with Supabase storage
- **Type Safety**: Built with TypeScript for robust, maintainable code
